# MDX-Doku. Bun-nativ. Du baust das Theme.

bundoc liefert die Content-Pipeline, den Router, die Suche und ein
kleines Set an Hooks. **Du** lieferst das Theme — jedes gerenderte
Pixel ist dein React-Code, kein blackbox-Skin.

```bash
bunx @clusterize/bundoc init meine-docs
cd meine-docs && bun install && bunx @clusterize/bundoc dev
```

## Schnellstart

Bearbeite eine beliebige `.mdx`-Datei in `content/` und die Seite lädt
per HMR neu. Das ist der ganze Loop.

<Callout type="info" title="Hinweis">
  Die englische Version dieser Seite ist die kanonische Quelle.
  Auf Seiten ohne deutsche Übersetzung zeigt bundoc das Original mit
  einem Banner.
</Callout>
