# Installation

bundoc ships as a single CLI. Add it to your project:

```bash
bun add -d bundoc
```

## Set up a project

bundoc expects a `bundoc.config.ts`, a `content/` directory, and a `theme/` directory.

| Path | Purpose |
| --- | --- |
| `bundoc.config.ts` | Locales, paths |
| `content/` | Your MDX files |
| `theme/index.tsx` | Your React theme |

Run the dev server:

```bash
bunx bundoc dev
```

## Project layout

A minimal site looks like this:

- `bundoc.config.ts`
- `content/index.mdx`
- `theme/index.tsx`

That's the whole contract.
